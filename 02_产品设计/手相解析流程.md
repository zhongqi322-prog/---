# 手相解析流程

## 目标

提供手相图片上传 mock 流程和传统相术角度的示例报告。

## 用户输入

- 左手 / 右手照片，第一版 mock
- 性别
- 年龄段
- 当前最想问的问题

## 流程

1. 用户阅读隐私提示。
2. 上传或选择 mock 手相图片。
3. 选择关注问题。
4. 展示免费摘要。
5. 解锁完整报告。

## 报告结构

- 掌纹基础观察
- 生命线 / 智慧线 / 感情线说明
- 传统相术依据占位
- 白话解释
- 现实建议
- 隐私提醒

## 隐私边界

- 第一版不做真实图像识别。
- 图片仅用于本次 mock 分析。
- 不做健康诊断。
- 不判断疾病。
