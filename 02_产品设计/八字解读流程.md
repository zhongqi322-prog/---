# 八字解读流程

## 目标

基于用户出生资料、当前问题和 mock 八字结构，生成带古籍出处占位和风险提醒的文化参考报告。

## 用户输入

- 称呼
- 性别
- 出生日期
- 日期类型：公历 / 农历 / 不确定
- 出生时间
- 出生地点
- 想问的问题
- 一句话描述困境

## 流程

1. 用户进入八字解读页。
2. 系统说明数据用途与隐私边界。
3. 用户填写资料。
4. 系统展示输入确认页。
5. 生成免费摘要。
6. 用户可选择 mock 支付解锁完整报告。
7. 展示完整报告并保存到我的记录。

## 报告模块

- 简要结论
- 出生资料整理
- 八字基础结构
- 五行与十神分析
- 当前问题分析
- 古籍出处占位
- 白话解释
- 现实建议
- 不建议做的事
- 风险提醒

## 古籍出处设计

P1 仅设计引用位置，不填真实原文。真实古籍片段进入 P3 时向用户索取。

## 边界

- 不输出必然命运。
- 不替代职业、医疗、法律或投资建议。
- 出生时间不确定时必须降低结论强度。
