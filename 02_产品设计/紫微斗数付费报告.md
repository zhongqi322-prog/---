# 紫微斗数付费报告

## 定位

紫微斗数作为八字之外的付费增强内容，用于展示更细分的人生领域分析。

## 免费摘要

- 命盘资料确认
- 当前问题方向
- 1-2 条简要趋势提醒
- 风险边界

## 完整报告

- 命盘基础信息
- 命宫 / 身宫解读
- 当前问题相关宫位
- 主星组合分析
- 四化与三方四正
- 大限 / 流年趋势占位
- 古籍出处占位
- 白话解释
- 现实建议
- 风险提醒

## 付费触发点

用户提交资料并看到免费摘要后，出现“解锁完整报告”按钮。

第一版只使用 mock 支付弹窗，不接真实支付。

## 价格展示

P1 仅设计展示位：

- 单份完整报告
- 八字 + 紫微联合增强报告
- 后续可扩展套餐

## 边界

- 不承诺准确预测未来。
- 不输出“必定结婚、必定分手、必定发财”等断语。
- 真实古籍出处进入 P3 后接入。
