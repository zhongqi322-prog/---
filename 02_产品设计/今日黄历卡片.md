# 今日黄历卡片

## 定位

今日黄历用于增强每日访问和传统文化氛围，不作为独立主项目。

## 展示内容

- 日期
- 今日宜
- 今日忌
- 今日五行
- 今日冲煞
- 今日吉时
- 今日提醒

## 交互

- 首页默认展示。
- 用户可点击“查看适合我的建议”，进入相关服务入口。
- 第一版使用本地 mock 数据。

## 文案边界

允许：

- “传统黄历角度可参考”
- “今日提醒”
- “适合安排相对稳妥的事项”

禁止：

- “今天必定发财”
- “今天一定出事”
- “必须按此行动”

## mock 数据字段

| 字段 | 说明 |
|---|---|
| `date` | 日期 |
| `yi` | 今日宜 |
| `ji` | 今日忌 |
| `wuxing` | 五行 |
| `chongsha` | 冲煞 |
| `goodHours` | 吉时 |
| `note` | 今日提醒 |
