# 传统术数古籍 AI 解读平台

一个基于中国传统术数古籍、规则算法与 AI 报告生成的中文玄学咨询平台。

项目核心不是让 AI 无依据地“算命”，而是结合古籍出处、结构化规则和用户资料，生成有依据、有解释、有边界的文化参考报告。

## 当前阶段

第 0 阶段：建立项目规则、产品范围、古籍知识库规范和发布路线。

当前阶段只创建文档与目录，不编写网站代码。

## 唯一项目系统

Obsidian、GitHub 与 Codex 必须操作同一套文件：

```text
D:\工作\2026\老祖宗-玄学\AI-Projects\fengshui-ai-site
```

- Obsidian：打开和编辑该目录。
- GitHub：该目录连接的唯一远程仓库，是正式事实源。
- Codex：只读取和修改该目录。

详见 [[00_项目总纲/系统一致性原则]]。当前本地 Git 已初始化，GitHub 远程仓库尚待连接。

## 第一版目标

- 可访问、可点击、可填写的中文网站
- 使用本地 mock 数据生成报告
- 每份报告展示古籍出处与风险提醒
- 提供 mock 支付解锁和轻量本地记录
- 支持移动端访问并最终发布至 Vercel

## 第一版功能
