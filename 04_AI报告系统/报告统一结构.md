# 报告统一结构

## 目标

所有功能报告使用统一结构，方便 P2 前端展示和 P4 推理接入。

## 统一结构

1. 简要结论
2. 用户资料整理
3. 当前问题重述
4. 分析结果
5. 古籍依据
6. 原文片段
7. 白话解释
8. 与用户问题的关系
9. 现实建议
10. 不建议做的事
11. 可继续追问
12. 风险提醒

## 引用块格式

```text
出处：待接入
书名：待接入
章节：待接入
原文：待用户提供古籍内容后接入
白话解释：待审核后接入
```

## P1 规则

- 只设计结构和占位。
- 不编造古籍原文。
- 不生成真实推理结论。
- 不接真实 AI。

## P2 前端展示需求

- 引用块独立展示。
- 风险提醒固定展示。
- 免费摘要与完整报告有清晰差异。
