# 00_当前项目状态

更新时间：2026-07-09

## 1. 项目定位

本项目是中文版“传统术数古籍 AI 解读平台”MVP。

核心定位不是让 AI 无依据算命，而是基于古籍出处、结构化规则、传统术数体系、AI 白话解释和风险边界，为用户生成有出处、有解释、有边界的文化参考报告。

## 2. 当前事实源

- GitHub 仓库是唯一真实数据源。
- Obsidian 只是项目大脑和编辑界面，不是第二套知识库。
- Codex 是执行开发者，不维护独立知识体系。
- 新会话默认先读本目录 5 个文件：`PROJECT_CONTROL/*.md`。
