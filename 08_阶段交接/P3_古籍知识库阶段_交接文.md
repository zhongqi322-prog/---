## 5. 当前有效决策

- 只做中文版 MVP，英文版暂停。
- GitHub 是唯一真实数据源，Obsidian 是同一项目目录下的编辑界面。
- P3 当前首批结构化古籍条目可用于后续 mock 报告引用。
- P4 只能读取 `copyright_status = 可使用` 且 `review_status = 已审核` 的古籍片段。
- P4 报告必须包含古籍出处、原文片段、白话解释、与用户问题的关系和风险提醒。
- 不得编造古籍原文、书名、章节或出处。
- P4 仍不接真实 AI、不接真实支付。
